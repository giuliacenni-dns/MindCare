# MindCare - Reaction Time Test Game

A simple reaction-time game that records your performance together with self-reported mood and sleep quality, and visualises trends across sessions.

## Requirements

- Python 3.6 or higher
- Required Python packages:
  - pygame
  - matplotlib

## Project Files (Important)

This project uses an external audio file during runtime:

- `ES_Feelings - Ryan James Carr.wav`

Make sure the audio file is in the **same folder** as `MindCare.py` (it is included in the repository). If the audio file is missing, the program may raise a “file not found” error when trying to load it.

## Installation

1. Make sure you have Python 3.6 or higher installed.

2. Install dependencies:

   ```bash
   pip3 install -r requirements.txt
   ```

   Or on Windows:

   ```bash
   pip install -r requirements.txt
   ```

## How to Run

1. Open Terminal (Mac/Linux) or Command Prompt (Windows).

2. Navigate to the folder containing `MindCare.py`:

   ```bash
   cd path/to/folder/with/MindCare.py
   ```

3. Run the game:

   ```bash
   python3 MindCare.py
   ```

   Or on Windows:

   ```bash
   python MindCare.py
   ```

## Game Instructions

1. Click "Click here to start" on the welcome screen
2. Select your current mood (Sad, Happy, Angry, Disgusted, Afraid, or Restless)
3. Select your sleep quality (Excellent, Good, Not so good, or Bad)
4. Click "PLAY" to start the game
5. Move your mouse over the white square as quickly as possible when it appears
6. The game lasts 30 seconds
7. After the game, check your results and (optionally) open the graphs

## Output Files

- `follow_square_daily.csv` — created automatically after the first completed session; stores all session data (date, username, mood, sleep, hits, spawns, mean reaction time)
- Graph image files generated by the program (saved in the project folder)

## Troubleshooting

- **If the game window doesn't appear**: run from Terminal/Command Prompt (not from an environment that blocks Pygame windows).
- **If you get “module not found”**: install dependencies using the commands above.
- **If you get “file not found” for the audio**: confirm `ES_Feelings - Ryan James Carr.wav` is in the same folder as `MindCare.py`.
